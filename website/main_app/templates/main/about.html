Мы - Upschedule

Upschedule это многофункциональный сервис для доступа к расписанию в школе.
Он включает в себя Telegram бота и веб интерфейс для более подробной информации

View on GitHub

Что сейчас?

Охват

База расписания бота охватывает классы с 8 по 11 включительно

Пользователи

На текущий момент зарегистрировано всего 60 пользователей, из них активны как минимум 20
(хотелось бы больше)

Функционал

От менеджера групп, который может остановить любой конфликт одной командой, до личного мотиватора

Что по технологиям?

Telegram

В качестве основы был выбран Telegram. Он быстрый, удобный и предоставляет широкие возможности для работы с ботами

Aiogram

Ассинхронный фреймворк для Telegram. Помогает делать ботов быстрее и проще

PostgreSQL

Самая лучшая реляционная база данных.
Ну не MySQL же...
